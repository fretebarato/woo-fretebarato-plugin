Plugin de integração de cotação de frete da plataforma Frete Barato com WooCommerce.

### Descrição

O plugin da Frete Barato para WooCommerce conecta sua loja às principais transportadoras do Brasil com cálculo automático de frete no checkout.

### Instalação

1. Instale o plugin através do WordPress ou faça upload dos arquivos para `/wp-content/plugins/woo-fretebarato-plugin/`
2. Ative o plugin
3. Vá em WooCommerce > Configurações > Entrega > Frete Barato
4. Configure seu Customer ID e CEP de origem
5. Crie uma conta no [painel Frete Barato](https://www.fretebarato.com) para obter seu Customer ID

### Requisitos

- WooCommerce instalado
- Customer ID da Frete Barato
- Produtos com peso configurado
